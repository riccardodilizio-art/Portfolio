ISTRUZIONI PER LE FOTO PERSONALI
==================================

Inserisci le tue foto in questa cartella con i seguenti nomi:

1. profile.jpg (o .png/.webp)
   -> Foto profilo principale per la sezione Hero
   -> Consigliato: primo piano professionale, formato quadrato (~800x800px)

2. about.jpg (o .png/.webp)
   -> Foto per la sezione "Chi Sono"
   -> Consigliato: foto a mezzo busto o in contesto lavorativo (~800x1000px)

3. event-1.jpg (o .png/.webp)
   -> Foto di un evento di cronometraggio sportivo
   -> Consigliato: foto durante un evento, formato landscape (~1200x800px)

4. event-2.jpg (o .png/.webp)
   -> Seconda foto evento (opzionale)
   -> Consigliato: altra foto di un evento diverso (~1200x800px)

5. abudhabi.jpg (o .png/.webp)
   -> Foto esperienza PCTO Abu Dhabi (opzionale)
   -> Consigliato: foto durante l'esperienza internazionale (~1200x800px)

FORMATI SUPPORTATI: .jpg, .jpeg, .png, .webp
DIMENSIONE CONSIGLIATA: max 500KB per foto (ottimizza con tinypng.com)
